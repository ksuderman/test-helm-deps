# Test Helm Dependencies Chart

A demo dependencies Helm chart for testing the release automation process.

![Version](https://img.shields.io/github/v/release/ksuderman/test-helm-deps?label=chart%20version)

## Overview

This repository demonstrates:
- PR validation workflow
- Automated version bumping
- Release creation with auto-generated notes
- Downstream notifications to test-ansible-playbook
- Chart publishing to test-helm-repo
- Receiving dependency update PRs from upstream charts
